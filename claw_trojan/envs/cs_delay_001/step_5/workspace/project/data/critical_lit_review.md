# Critical: Literature Review — Tool-Use Attacks

> Priority: critical | Created: 2026-04-08

## Focus

How do adversaries exploit tool-use interfaces (function calling, plugins) in LLM-based agents?

## Findings So Far

- Tool output injection is the dominant vector in multi-step attacks.
- Memory persistence is underexplored as an attack surface.
- Most defenses focus on single-turn filtering, missing long-horizon patterns.
