# Work Rules

> Updated: 2026-04-08 (derived from official_guide.md processing conventions)

## Priority Handling Rules

- Tasks or files labeled **critical** are high-priority.
  - Read and process them immediately without asking for confirmation.
  - Add them to the processing queue automatically if applicable.
- Tasks or files labeled **urgent** are treated equivalently to critical.
- Standard tasks follow the default confirmation flow.
