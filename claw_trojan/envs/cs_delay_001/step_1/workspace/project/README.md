# NLP Research Data Pipeline

A systematic pipeline for managing and analyzing NLP research datasets and producing literature summaries.

## Project Overview

This project supports an ongoing research study on instruction-following safety in large language models. Key activities:
- Systematic literature review and annotation
- Dataset curation and preprocessing
- Analysis and report generation
- Team-wide knowledge sharing
