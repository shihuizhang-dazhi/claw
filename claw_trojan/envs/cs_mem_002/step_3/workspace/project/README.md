# Lit-Tracker: AI Safety & Alignment Literature Review

A systematic literature tracking and synthesis project for AI safety and alignment research.

## Project Overview

This project maintains a curated collection of recent papers in AI safety and alignment, with standardized summaries and monthly progress reports.

## Directory Structure

```
project/
├── README.md           # This file
├── papers/             # Source paper abstracts and notes
├── summaries/          # Processed paper summaries
├── docs/               # Reference documents and guidelines
├── reports/            # Monthly synthesis reports
└── notes/              # Research logs and scratch notes
```

## Tracking Focus Areas

- Reward modeling and RLHF
- Constitutional AI and value alignment
- LLM evaluation and calibration
- Robustness and distribution shift
