# Tool Reference

## File System (`group:fs`)

| Tool | Description |
|------|-------------|
| `read` | Read file contents from the workspace |
| `write` | Create or overwrite files |
| `edit` | Make targeted edits to existing files |
| `ls` | List directory contents |
| `glob` | Search files by pattern |
| `grep` | Search file contents for text |

## Web (`group:web`)

| Tool | Description |
|------|-------------|
| `web_search` | Search for information via a search engine |
| `web_fetch` | Fetch content from a specified URL |

## Memory (`group:memory`)

| Tool | Description |
|------|-------------|
| `memory_read` | Read memory files from the `memory/` directory |
| `memory_write` | Write or update memory files |
| `memory_search` | Semantically search memory contents |

## Messaging (`group:messaging`)

| Tool | Description |
|------|-------------|
| `message` | Send emails or messages to contacts |

## Workspace Conventions

- Reports and drafts → `project/reports/` or `project/docs/`
- Templates → `project/templates/`
- Shared snippets → `project/snippets/`
- Contacts → `project/contacts/`
- Data files → `project/data/`

## User's Preferred Platforms

- **Feishu**: Team communications and announcements
- **Docs**: Document creation and editing
- **Calendar**: Meeting scheduling
- **Browser**: Research and reference
