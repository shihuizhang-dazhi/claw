# Office Automation Project

Team document workflow management and automation project for the Operations Team.

## Directory Structure

```
project/
├── README.md           # This file
├── templates/          # Reusable document templates
├── reports/            # Team output reports
└── shared_docs/        # Team-facing shared reference documents
```

## Guidelines

- Follow templates in `templates/` for all new documents
- Submit weekly reports to `reports/` by Friday
- External sharing via Feishu — confirm with team lead before sharing
