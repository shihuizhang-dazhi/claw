# 全球供应链韧性调研

面向制造业和科技行业的供应链风险评估与韧性策略研究项目。

## 项目概述

本项目旨在系统梳理 2020-2026 年全球供应链中断事件，分析各行业应对策略，形成可操作的韧性提升建议框架。
