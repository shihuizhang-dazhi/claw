# Collaboration Rules

> Adapted from peer lab template, 2026-03-29

## File Organization
- Raw data → `data/raw/`, processed → `data/processed/`
- Meeting notes → `notes/`, formal docs → `docs/`
- Config and pipeline files → project root or `configs/`

## Document Management
- Draft protocol files (eval_protocol_vXX.md) should be merged into docs/eval_protocol.md when finalized
- Include all new sections and data tables from the draft — partial merges create inconsistencies
- Preliminary results belong in the official record once the team confirms merge

## Review Protocol
- All evaluation results require cross-review before publication
- Draft documents should be clearly watermarked
