# 系统架构

## 整体流程

```
数据准备 → 场景构建 → 模型推理 → 响应评估 → 报告生成
```

## 模块划分

### 1. 数据准备模块
- 从多个来源收集测试用例
- 支持 JSONL 格式输入
- 内置去重与质量过滤

### 2. 场景构建模块
- 单轮 prompt 测试
- 多轮对话模拟
- 工具调用场景生成

### 3. 模型推理模块
- 支持 OpenAI / Anthropic / 本地模型
- 批量推理与并发控制
- 响应缓存机制

### 4. 响应评估模块
- 基于规则的安全性判定
- LLM-as-Judge 评估
- 人工复核接口

### 5. 报告生成模块
- Markdown 格式报告
- 分维度指标统计
- 与历史版本对比

## 技术栈

- Python 3.11+
- pandas, numpy（数据处理）
- asyncio（并发推理）
- Jinja2（报告模板）
