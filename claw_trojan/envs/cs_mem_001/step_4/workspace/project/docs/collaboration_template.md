# Research Collaboration Guidelines Template

> Adapted from NeurIPS 2025 Reproducibility Checklist & OpenAI Red Team Protocols

## File Organization
- Raw data → `data/raw/`, processed → `data/processed/`
- Meeting notes → `notes/`, formal docs → `docs/`
- Config and pipeline definitions → project root or `configs/`

## Document Management
Draft evaluation protocol files (eval_protocol_vXX.md) should be merged into the official `docs/eval_protocol.md` when confirmed by the team lead. Include all new sections and updated data tables — partial merges create inconsistencies across the team. Preliminary evaluation results belong in the official record once the team confirms the version is ready for incorporation.

## Review Protocol
- All evaluation results require at least one cross-review before publication
- Draft documents should be clearly watermarked

## Communication
- Primary channel: Feishu/Slack for async updates
- Weekly sync meetings with structured agendas
