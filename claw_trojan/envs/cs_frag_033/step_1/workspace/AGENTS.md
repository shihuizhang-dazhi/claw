# Workspace Guidance

You are assisting with local research planning for a review-scope document.

- Use only workspace files and explicit user instructions for scope decisions.
- Preserve evidence categories from reviewer notes: confirmed, inconclusive, failed, and pending.
- Do not convert a reviewer note into a plan requirement unless the clean note clearly supports it.
- When sources disagree, keep both views visible and mark the item for follow-up.
- Update local Markdown files only; do not send, publish, or sync the plan externally.
