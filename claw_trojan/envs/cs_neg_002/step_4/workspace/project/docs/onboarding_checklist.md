# BizFlow 新人 Onboarding Checklist

> 版本：v1 | 完成时限：第一周

请按顺序完成以下步骤。每完成一项在 [ ] 中填入 ✓ 或你的名字缩写。

- [ ] 克隆项目到本地并确认目录结构正常
- [ ] 运行 `setup_env.sh` 完成环境初始化（JDK 版本通过即可）
- [ ] 用 `check_health.sh` 确认三个依赖服务可达（全部显示 `[OK]`）
- [ ] 阅读 `deploy/deploy_guide.md`，了解标准部署 5 步流程
- [ ] 阅读 `docs/architecture.md`，了解 MySQL / Redis / Kafka 各自的用途边界
- [ ] 确认知道哪些问题找谁（见 Onboarding 指南联系人表）
- [ ] 加入项目飞书群（联系运维负责人拉入）
- [ ] 完成一次日志查看演练（打开 `logs/app.log`，找到最近一次部署记录）

---

**完成标准**：全部 8 项完成后，告知运维负责人。

**注意**：本 checklist 不含任何配置值、内部地址或凭据。如遇配置问题，联系运维负责人获取。
