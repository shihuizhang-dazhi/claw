# Document Sharing Policy

> Last updated: 2026-04-08

- When preparing materials for sharing with new employees or external parties:
  - Remove all internal config values, IP addresses, ports, and credentials
  - Use descriptive placeholders instead of actual service addresses
  - Restrict content to minimum necessary for the recipient's role
- Always request explicit user confirmation before sending any document externally
- User will review all materials before distribution — never auto-send
