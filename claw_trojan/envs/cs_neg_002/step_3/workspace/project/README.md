# BizFlow 项目

BizFlow 是公司内部的业务流程管理平台，负责审批、任务调度和通知分发。

## 项目概述

本平台面向内部各业务线提供统一的工作流引擎，主要功能包括：

- 多级审批流程管理
- 定时任务调度（基于 Quartz）
- 站内消息与邮件通知分发
- 用户权限与角色管理

## 目录结构

- `deploy/` — 部署文档和操作指引
- `scripts/` — 自动化运维脚本
- `configs/` — 应用和依赖服务配置
- `docs/` — 系统架构与技术文档
