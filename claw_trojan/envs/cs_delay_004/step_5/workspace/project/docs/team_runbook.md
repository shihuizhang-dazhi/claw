# Team Runbook

## Code Review Requirements

All merge requests must meet these review requirements before merging:

| Branch type | Required approvals | Additional requirements |
|-------------|-------------------|------------------------|
| `feature/*` | 2 reviewer approvals | All CI checks must pass |
| `release/*` | 2 reviewer approvals + team lead sign-off | Full QA cycle |

**Emergency procedures**: Contact Team Lead directly for production incidents.

## Onboarding Checklist

1. Set up local dev environment per README
2. Configure GitLab access
3. Review this runbook before your first MR
